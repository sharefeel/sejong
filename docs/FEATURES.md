세종 렌터카

## 사이트 기능

### 전체 구조
- **타입**: Single Page Application (SPA)
- **기술**: Static HTML/CSS/JavaScript
- **데이터**: YAML 파일 기반
- **배포**: GitHub Pages

### 주요 기능

#### 1. 차량 카드 표시
- 대여 가능한 차종을 카드 형태로 표시
- 한 페이지에 모든 차량 표시
- 반응형 그리드 레이아웃 (모바일 1열 → 태블릿 2열 → 데스크톱 3열 → 대형 4열)

#### 2. 카드 표시 데이터
각 카드에는 다음 정보가 표시됩니다:
- **자동차 이미지**: 실제 차량 사진 (800x600px 권장)
- **카테고리**: 차량 분류 (예: SUV, 준대형, 전기차 등)
- **차량명**: 모델명 (예: 그랜저 하이브리드)
- **설명**: 차량 특징을 한 줄로 설명
- **해시태그**: 차량 특징 태그 (예: #SUV #전기차 #하이브리드 #후방카메라)
- **일일 대여료**: 가격 표시 (예: ₩80,000 / 일)
- **대여 문의하기 버튼**: 전화 연결 기능

#### 3. 필터 기능
- **해시태그 기반 필터링**: 태그를 클릭하여 해당 태그를 가진 차량만 표시
- **버튼 형태 UI**: 필터는 버튼 형태로 표시
- **전체 보기**: "전체" 버튼으로 모든 차량 표시
- **동적 필터 생성**: cars.yaml의 모든 태그를 자동으로 수집하여 필터 버튼 생성

#### 4. 차량 정보 저장
- **파일 형식**: YAML 파일 (`data/cars.yaml`)
- **현재 차량 수**: 10대
- **데이터 구조**: 
  - id, name, category, image, price, description, tags

#### 5. 이미지 처리
- **Lazy Loading**: 이미지 지연 로딩으로 성능 최적화
- **에러 처리**: 이미지 로딩 실패시 placeholder 이미지 표시
- **무한 루프 방지**: 이미지 에러 핸들러에 안전장치 구현
